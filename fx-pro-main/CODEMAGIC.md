# Codemagic build

This repository is ready for Codemagic with `codemagic.yaml` at the repository root.

## Required Codemagic setup

Create an environment group named `fxpro_env` with:

- `EXPO_PUBLIC_BACKEND_URL`: production FastAPI URL, for example `https://api.example.com`
- `EXPO_PUBLIC_FIREBASE_VAPID_KEY`: Firebase Web Push VAPID key
- `EXPO_PUBLIC_EAS_PROJECT_ID`: Expo project id if you use Expo push tokens

For iOS, connect App Store Connect in Codemagic and name the integration `codemagic`, then enable automatic signing for bundle id `com.fxpro2026.app`.

For Android, upload the release keystore in Codemagic as `fxpro_android_keystore`.

## iOS native project

The iOS `.xcodeproj` and `.xcworkspace` are generated by Codemagic on macOS with:

```sh
cd frontend
npx expo prebuild --platform ios --clean
cd ios
pod install
```

iOS prebuild is not supported on Windows, so the repo keeps the generated iOS project out of source control and lets Codemagic create it during the build.
